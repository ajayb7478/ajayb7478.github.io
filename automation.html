Welcome to My Automation Project

This project is designed to streamline and automate various tasks, enhancing efficiency and productivity. This project was made to automate test cases for a demo ecommerce Website

github repository link

Key Features:

  • Automated test cases
  • Using C#
  • Using Specflow BDD

Test Cases

  • Login Functionality:
    1. Valid Login: Test with correct username and password.
    2. Invalid Login: Test with incorrect username and/or password.
    3. Empty Fields: Test login with both fields empty.
    4. Username Only: Test login with only the username filled.
    5. Password Only: Test login with only the password filled.
  • Navigation:
    1. Home Page Access: Check if the home page loads correctly after login.
    2. Product Page Access: Verify navigation to product pages.
    3. Cart Page Access: Verify navigation to the cart page.
